#연산자

#### 특징
> 비교 연산자는 boolean 값으로 반환한다  
> Primitive 타입의 경우 [[Constant pool_작성안함]] 내의 주소를 비교하고.
> Reference 타입의 경우 오브젝트의 주소를 비교한다.


#### 정수, 실수 비교 (데이터 값)
|구분|연산식|설명|
|---|---|---|
|동등비교|\==|두개의 값이 같은지 비교|
||!=|두개의 값이 다른지 비교|
|크기비교|>|왼쪽이 크면 true|
| |>=|왼쪽이 크거나 같으면 true|
||<|오른쪽이 크면 true|
||<=|오른쪽이 크거나 같으면 true|


#### 주의사항
> float 타입과 double 타입을 비교하게 될 경우  
> 정밀도 차이에 의해 같은 값으로 보여도 false를 반환한다  
> 예시 :  
> 0.1f == 0.1 --> false 반환  
> 그리하여 강제 타입 변환 후 비교연산을 하여야 한다  
> 수정후 예시 :  
> 0.1f == (float) 0.1 --> true
