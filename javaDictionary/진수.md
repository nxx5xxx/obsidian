## 2진수
> 0b 또는 0B 로 시작하고 0과 1로 작성  
> 예시 :  
> int x = 0b1011;  
> int y = 0B10111;
### 2진수를 10진수로 계산하기
> int x = 0b1011 이라면  
> 1011
> 1\*2^3 + 0\*2^2 + 1\*2^1 + 1\*2^0  
> 8 + 0 + 2 + 1 = 11 이렇게 계산할 수 있다

## 8진수
> 0으로 시작하고 0~7 숫자로 작성  
> 예시 :  
> int x = 070115;  

### 8진수를 10진수로 계산하기
> int x = 070115;  이라면  
> 70115
> 7 \* 8^4 + 0 \* 8^3 + 1 \* 8^2 + 1 \* 8^1 + 5 \* 8^0  
> 28,672 + 0 + 64 + 8 + 5 = 28,749  
> 다른 진수들도 이런식으로 변환이 가능하다

### 16진수
> 0x또는 0X로 시작하며 0~9, A~F 또는 a~f로 작성한다
