## 자동 타입 변환 (promotion)
> 묵시적으로 자동으로 타입이 변환 되는 것  
> 값의 허용 범위가 작은 타입이 허용 범위가 큰 타입으로 대입 될 때 발생한다.  
> 예시 :   
> byte → short  
> char → int → long → float → double  

#### 사용예시
>byte byteValue = 30;  
>int intValue = byteValue; <여기서 자동 타입 변환이 되는것
